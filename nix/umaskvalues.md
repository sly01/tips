    Octal value: Permission
    0:	read,write and execute
    1:	read and write
    2:	read and execute
    3:	ready only
    4:	write and execute
    5:	write only
    6:	execute only
    7:	no permission

    Example of umask 022
    For files permission
    666 - 022 = 644
    For directory permission
    777 - 022 = 755
